# SysCore Examples Catalog

Example configurations demonstrating API utilization:

- `examples/process/basic_fork.c`: Fork process creation.
- `examples/ipc/anonymous_pipe.c`: Pipe communication loop.
- `examples/ipc/named_fifo.c`: Bounded named fifo communication.
- `examples/threading/basic_thread.c`: Multi-thread creation and joining.
- `examples/sync/mutex_demo.c`: Mutex critical section locking.
- `examples/memory/shared_memory_demo.c`: Parent/child shared memory mapping.
- `examples/ipc/message_queue_demo.c`: Standard POSIX Message Queue wrapper.
- `examples/memory/mmap_anonymous_demo.c`: Page-aligned anonymous memory mapping.
